## Community libraries 
### MOKO
* [icerockdev/moko-widgets](https://github.com/icerockdev/moko-widgets) - multiplatform UI declarative library (allows you to create ios and android application using only shared code in Kotlin);
* [icerockdev/moko-mvvm](https://github.com/icerockdev/moko-mvvm) - architecture components MVVM for mobile multiplatform + LiveData operators;
* [icerockdev/moko-network](https://github.com/icerockdev/moko-network) - ktor-client api generation from OpenAPI specification;
* [icerockdev/moko-resources](https://github.com/icerockdev/moko-resources) - work with localization resources from shared code;
* [icerockdev/moko-permissions](https://github.com/icerockdev/moko-permissions) - access for runtime permissions from shared code;
* [icerockdev/moko-media](https://github.com/icerockdev/moko-media) - get photo/video from camera and gallery from the shared code;
* [icerockdev/moko-fields](https://github.com/icerockdev/moko-fields) - component for forms with validations;
* [icerockdev/moko-units](https://github.com/icerockdev/moko-units) - RecyclerViewAdapter/UITableViewDataSource/UICollectionViewDataSource with filling from shared code, delegate datasource;
* [icerockdev/moko-graphics](https://github.com/icerockdev/moko-graphics) - classes for work with graphics in mpp;
* [icerockdev/moko-parcelize](https://github.com/icerockdev/moko-parcelize) - @Parcelize support in common code;
* [icerockdev/moko-time](https://github.com/icerockdev/moko-time) - time and timers in shared code;


### Trikot
* [mirego/trikot.foundation](https://github.com/mirego/trikot.foundation) - misc usefull components - timers, data works, mpp AtomicReference, freeze and quques for multithreading;
* [mirego/trikot.http](https://github.com/mirego/trikot.http) - http-client, usinf kotlinx.serialization. The ktor-client analog;
* [mirego/trikot.kword](https://github.com/mirego/trikot.kword) - multiplatform localization using [Accent](https://www.accent.reviews/);
* [mirego/trikot.metaviews](https://github.com/mirego/trikot.metaviews) - abstract UI from shared code for data binding;
* [mirego/trikot.streams](https://github.com/mirego/trikot.streams) - reactive streams with multithreading;

### Korlibs
* [korlibs/klogger](https://github.com/korlibs/klogger) - logging tool from shared code;
* [korlibs/klock](https://github.com/korlibs/klock) - date and time work including timezones and formatting;
* [korlibs/kbignum](https://github.com/korlibs/kbignum) - bigint, bigdecimal;
* [korlibs/krypto](https://github.com/korlibs/krypto) - secureRandom, sha1, sha256, md5, hash, aes;
* [korlibs/korim](https://github.com/korlibs/korim) - image files handling (supporting different formats) from shared code;
* [korlibs/korau](https://github.com/korlibs/korau) - audio files handling (supporting different formats) from shared code;
