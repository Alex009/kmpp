## Useful plugins 
* [icerockdev/mobile-multiplatform-gradle-plugin](https://github.com/icerockdev/mobile-multiplatform-gradle-plugin) - plugin for make mobile multiplatform projects setting as simple as possible; android, but for shared code;
* [Kotlin/dokka](https://github.com/Kotlin/dokka) - documentation generator for kotlin code;
* [whyoleg/kamp](https://github.com/whyoleg/kamp) - pre-filled configuration for mpp projects;
* [Foso/MpApt](https://github.com/Foso/MpApt) - annotation processor for mpp.

### Configuration
* [gmazzo/gradle-buildconfig-plugin](https://github.com/gmazzo/gradle-buildconfig-plugin) - configuration generator;
* [yshrsmz/BuildKonfig](https://github.com/yshrsmz/BuildKonfig) - configuration class generator like a BuildConfig in android;
* [Karumi/Hagu](https://github.com/Karumi/Hagu) - one more configuration generator;

### Coroutines
* [andrewemery/recast](https://github.com/andrewemery/recast) - callback functions generations from suspend functions in kotlin/native;
* [feilfeilundfeil/kotlin-native-suspend-function-callback](https://github.com/feilfeilundfeil/kotlin-native-suspend-function-callback) - callback functions generator from suspend function in kotlin/native.
